1С-Обмен для WooCommerce: полная интеграция с 1С:Предприятие, СБИС и МойСклад по протоколу CML2/CML3.

== Description ==

**1C-Integration-WooCommerce** — плагин для WordPress, реализующий двусторонний обмен данными между WooCommerce и системами учёта на базе 1С:Предприятие 8 (а также СБИС и МойСклад).

Плагин работает с **любой конфигурацией 1С**, которая поддерживает обмен по протоколу CML2 / CML3.

= Основные возможности =

* Загрузка и обновление категорий товаров (с поддержкой вложенности)
* Загрузка и обновление свойств/атрибутов товаров и их значений
* Загрузка и обновление товаров, артикулов, описаний
* Загрузка и обновление изображений товаров (с хеш-проверкой — обновление только при изменении)
* Загрузка и обновление цен (базовая и распродажная) и остатков
* Поддержка полного и инкрементального обмена (только изменения)
* Передача файлов частями (для обхода ограничений хостинга)
* Пошаговая обработка с контролем времени (работает даже на shared-хостинге)
* Поддержка приёма данных в ZIP-архиве (ускоряет обмен)
* Выгрузка заказов WooCommerce в 1С в режиме реального времени
* Гибкий выбор типов цен и складов
* Поиск существующих товаров по артикулу (SKU) или по 1С-идентификатору
* Подробные логи обмена с фильтрацией по уровню
* Тест подключения прямо из админки с диагностикой
* Система лицензирования (ключ формата WC1C-XXXX-XXXX-XXXX-XXXX)

= Поддерживаемые системы =

* **1С:Предприятие 8** — Управление торговлей, Розница, Комплексная автоматизация, ERP и любые другие конфигурации с поддержкой CML2/CML3
* **СБИС** — как источник обмена
* **МойСклад** — как источник обмена

= Требования =

* WordPress 5.8+
* WooCommerce 4.0+
* PHP 7.4+
* Расширения PHP: SimpleXML, mbstring (рекомендуется ZipArchive для архивной передачи)

== Installation ==

1. Загрузите ZIP-архив плагина через «Плагины → Добавить новый → Загрузить плагин».
2. Нажмите «Установить» и затем «Активировать».
3. Перейдите в меню **1C Integration → Активация**.
4. Введите лицензионный ключ или заполните форму запроса (укажите имя, email, сайт, соц. сети).
5. После активации перейдите в **1C Integration → Настройки** и задайте пользователя и пароль обмена.
6. Включите чекбокс «Включить обмен» и сохраните настройки.
7. Используйте URL обмена при настройке узла в 1С / СБИС / МойСклад.

= Настройка в 1С (пример для Управление торговлей 11) =

1. НСИ и администрирование → Интеграция с другими программами → Обмен с сайтом.
2. Создайте новый узел обмена.
3. В поле «Адрес» укажите URL обмена из настроек плагина.
4. Введите пользователя и пароль, заданные в плагине.
5. Запустите тестовый обмен.

== Frequently Asked Questions ==

= С какими конфигурациями 1С работает плагин? =

С любой конфигурацией 1С:Предприятие 8 (облачной и обычной), которая поддерживает обмен по протоколу CML2/CML3 с сайтом.

= Как получить лицензионный ключ? =

Перейдите в 1C Integration → Активация и заполните форму «Запросить лицензионный ключ». Разработчик рассмотрит запрос и пришлёт ключ на указанный email.

= Тест подключения показывает ошибку «Обмен не включён». Что делать? =

Зайдите в Настройки → включите чекбокс «Включить обмен», задайте пользователя и пароль, сохраните настройки, затем запустите тест снова.

= Работает ли со СБИС и МойСклад? =

Да. Оба продукта поддерживают обмен с сайтом по протоколу CML. Подробнее см. документацию СБИС и МойСклад.

== Privacy Policy ==

При запросе лицензионного ключа плагин отправляет на почту разработчика: имя, email, URL сайта и ссылку на соц. сети (если заполнено). Данные используются исключительно для выдачи и поддержки лицензии и третьим лицам не передаются.

В процессе активации плагин передаёт на сервер лицензий (рукодер.рф) URL сайта и версию плагина.
